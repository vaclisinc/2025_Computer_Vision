Part A.2: Recover Homographies

Approach

To warp images and create panoramic mosaics, we need to compute the homography matrix H that maps points from one image to another. Given correspondence points between two images, we solve for the 8 unknowns in the homography using a system of linear equations.

Mathematical derivation of the homography system

Mathematical Formulation

For each correspondence pair (x, y) ↔ (u, v), we construct two linear equations:

  • h₁x + h₂y + h₃ - h₇ux - h₈uy = u
  • h₄x + h₅y + h₆ - h₇vx - h₈vy = v

With n correspondence points, we build the system Ah = b where:

  • A is a 2n × 8 matrix containing the coordinates and their products
  • h is an 8 × 1 vector of unknown homography coefficients
  • b is a 2n × 1 vector of target coordinates

We solve this using least-squares: h = (AᵀA)⁻¹Aᵀb, then reconstruct the 3×3 homography matrix with h₉ = 1.

def computeH(pts1, pts2):
    n = len(pts1)
    A = np.zeros((2*n, 8))
    b = np.zeros((2*n, 1))

    for i in range(n):
        x, y = pts1[i]
        u, v = pts2[i]

        A[2*i] = [x, y, 1, 0, 0, 0, -u*x, -u*y]
        b[2*i] = u

        A[2*i+1] = [0, 0, 0, x, y, 1, -v*x, -v*y]
        b[2*i+1] = v

    h, _, _, _ = np.linalg.lstsq(A, b, rcond=None)

    H = np.zeros((3, 3))
    H[0, :] = h[:3].flatten()
    H[1, :] = h[3:6].flatten()
    H[2, :2] = h[6:].flatten()
    H[2, 2] = 1

    return H, A, b
